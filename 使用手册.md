# 简历网站使用说明（客户版）

写给第一次打开链接的用户。目标：**10 分钟内完成首版内容，并能分享给 HR 查看。**

---

## 0. 打开网站后先看什么

- 这是**单页网站**，顶部有 **首页 / 履历 / 作品** 三个分区，点一下即可跳转。
- 首次进入会自动播放**新手引导**；也可点右上角 **「使用说明」** 随时复习，或 **「重新播放新手引导」**。
- 若右下角有 **「智能导入简历」「站点编辑」**，说明你有**编辑权限**；没有则是**只读浏览**（适合 HR 查看）。

---

## 1. 推荐上手路径（9 步）

### 第 1 步：确认编辑权限

| 情况 | 你怎么做 |
|------|----------|
| 站长给了 **编辑链接** | 用编辑链接打开，即可编辑 |
| 打开后看不到编辑按钮 | 联系站长：换编辑链接，或把你的公网 IP 加入白名单 |
| 你是 HR，只有查看链接 | 只读浏览即可，顶栏会出现 **HR 速览** 条 |

### 第 2 步：智能导入简历（最快，强烈推荐）

1. 点右下角 **「智能导入简历」**
2. 上传 **PDF / Word（.doc/.docx）/ 纯文本 .txt**
3. 等待解析完成，在预览页查看识别到的字段与 **「建议核对」** 提示
4. 点 **「确认填入站点」**，系统会自动写入各区块并保存

**导入后建议重点核对：**

- 首页：姓名、岗位、联系方式、核心亮点
- 履历 → 工作经历 / 项目经历条数是否合理（同一项目不应被拆成多条）
- 履历 → 教育 → 点开卡片 → **校园成果 → 奖项荣誉**（每条应为「竞赛名 + 等级 + 日期」完整一行）
- 作品区：若简历里没有外链作品，需手动补 2 个以上

### 第 3 步：完善首页

两种方式任选或组合使用：

**A. 就地编辑（最快）**  
在首页直接点击文字，修改姓名、意向岗位、一句话介绍、核心亮点、邮箱 / 电话 / 其他联系方式。修改后会**自动保存**。

**B. 首屏与形象（较完整）**  
点 **「站点编辑 → 首屏与形象」**，可设置：

- 形象照 / 证件照
- 首屏辅助文案
- 联系二维码
- 右侧展示窗模式（重点展示 / 证件照等）

### 第 4 步：检查履历区

点击顶部 **「履历」**，分区包括：

| 分区 | 用途 |
|------|------|
| 工作经历 | 实习 / 正式工作：公司、职位、时间、关键成果 |
| 项目经历 | 课程设计、竞赛项目、科研等（含技术要点） |
| 教育背景 | 学校、专业、时间；点开可看 **校园成果**（学业、课程、校园经历、**奖项荣誉**） |

- 每条卡片可 **新增 / 编辑 / 删除**
- 点击卡片底部引导语或「查看详情」进入详情页
- **代表项目** 可添加图片、视频、代码、链接、文档等多种证明

### 第 5 步：补充作品区

点击顶部 **「作品」**：

- 每个作品建议：**标题 + 一句话价值 + 封面图 + 外链**
- 封面点击后在新标签页打开链接（GitHub、Behance、视频页等）
- 优先放**最近、最相关、能打开**的内容，不必堆砌数量

### 第 6 步：调整主题与背景

左下角 **「主题」**：

- 切换整站配色（墨 / 暖 / 编辑 / 海洋等）
- 选择页面背景：**纯色纸面** / **自定义图片** / **轻柔流光**
- 自定义图片可在弹窗中上传并调节强度

### 第 7 步：切换语言（如需中英双语）

右上角 **「EN / 中文」** 切换展示语言。  
若中英都要完整，请**分别切换**后检查首页、履历、作品是否都已填写。

### 第 8 步：预览自查

点右上角 **「预览」**：

- 隐藏所有编辑入口，模拟访客视角
- 检查排版、图片、视频、外链是否正常
- 再用手机打开看一遍（或缩小浏览器窗口）

### 第 9 步：分享给 HR / 客户

点右上角 **「分享」**：

- **复制链接**：发给对方（系统会自动去掉编辑密钥，避免泄露编辑权限）
- **二维码**：手机扫码打开
- **发送到邮箱**：可代发或调用本机邮件应用（取决于服务器是否配置发信服务）

**可选：** 右下角 **「链接访问记录」** 可查看只读链接的打开时间与大致地区，方便跟进 HR 是否已读。

---

## 2. 右下角编辑区说明

有编辑权限时，右下角固定有三个入口：

| 按钮 | 作用 |
|------|------|
| **智能导入简历** | 上传 PDF/Word，自动填入全站 |
| **链接访问记录** | 查看谁何时打开了你的只读链接 |
| **站点编辑** | 展开菜单：首屏与形象 / 履历页用词 / 作品页用词 |

**站点编辑** 适合改「不适合在页面上直接点改」的全局设置；具体经历与作品仍在 **履历 / 作品** 分区内编辑。

---

## 3. 右上角工具说明

| 按钮 | 作用 |
|------|------|
| **预览** | 模拟访客视角（仅有编辑权限时显示） |
| **EN / 中文** | 切换界面与内容语言 |
| **使用说明** | 打开本手册摘要；底部可重播新手引导 |

## 4. 数据保存与发布（你需要知道的）

- 编辑过程中内容会**自动保存**到当前浏览器。
- 若你使用的是站长分配的 **编辑链接**（带独立空间），保存后还会**自动同步到服务器**，HR 通过 **查看链接** 打开的是最新发布版。
- 换电脑、换浏览器或清除网站数据，**本机草稿可能丢失**；重要版本请及时分享链接备份。
- 若顶栏出现黄色 **保存失败** 提示（常见于图片过大），请压缩图片或换外链，再重试。

## 5. 媒体与外链建议

在 **首页 Showcase** 或 **履历代表项目** 中可添加：

| 类型 | 说明 |
|------|------|
| 图片 | 适合设计稿、海报、摄影；建议高清横图 |
| 视频 | YouTube / Bilibili 公开链接可嵌入播放 |
| 代码 | 粘贴关键代码片段 |
| 链接 | 跳转到仓库、作品页、小程序说明页等 |
| 文档 | PDF 展示效果最佳 |

## 6. 分享前检查清单

- [ ] 姓名、意向岗位、联系方式正确
- [ ] 至少有教育背景 + 若干工作 / 项目 / 作品
- [ ] 所有图片、视频、文档、外链能正常打开
- [ ] 已用「预览」和手机各看一遍
- [ ] 分享的是 **查看链接** 或去掉编辑参数的链接（勿把编辑链接发给 HR）

## 7. 常见问题

### 看不到编辑按钮

- 当前为只读模式：请用 **编辑链接** 打开，或联系站长添加 IP 白名单。

### 智能导入后内容不对

- 在对应分区手动修改；优先使用**文字版** PDF/Word，扫描件识别率较低。
- 关注导入预览里的 **「建议核对」** 列表。

### 校园成果里奖项被拆成两条

- 在 **教育背景 → 卡片详情 → 奖项荣誉** 中合并为一条；或重新导入后核对（系统会持续优化解析逻辑）。

### 分享链接对方打不开

- 本地 `localhost` 只有你自己能开；需使用正式域名或隧道地址。
- 确认链接未过期、未拼错。

### 页面突然无法点击

- 刷新页面；若刚关闭到一半的新手引导，刷新通常可恢复。

### 视频无法播放

- 确认链接为 YouTube / Bilibili **公开**页面地址。

---

## 8. 内容撰写建议（实战）

- 首页一句话：写「我能解决什么问题」，不要写成长篇自我介绍。
- 每条经历尽量有**量化结果**（提升 xx%、完成 xx 项、响应 < xx ms）。
- 作品求精不求多：**相关 + 可验证 + 能打开**。
- 改一点、预览一点，满意再分享。

## 附录：站长说明（编辑链接 / 查看链接）

若你购买的是「独立链接版」：

- **编辑链接（editUrl）**：客户用来编辑并自动发布
- **查看链接（viewUrl）**：客户发给 HR，只读且可记录访问

客户在编辑链接里保存后，HR 打开查看链接即可看到最新内容，无需客户手动「点发布」。

服务器创建空间方式见项目 README 与 `docs/DEPLOY-AUTO.md`（站长阅读，客户可忽略）。
