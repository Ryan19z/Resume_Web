# 对外承诺边界（智能导入 + 在线简历）

> 面向客户 / HR 的说明文案，与产品内「智能导入 → 查看服务承诺边界」一致。推广或签约前可直接引用。

---

## 我们提供什么

1. **独立链接**：EditURL（客户编辑）与 ViewURL（HR 只读），手机与电脑均可打开。
2. **在线发布**：编辑后自动同步到服务器，HR 打开 ViewURL 看到最新内容。
3. **HR 访问记录**：匿名记录是否被打开及大致地区，不保存访问者 IP 明文。
4. **智能导入**：从 PDF / Word / 文本快速生成**可编辑初稿**，减少手工录入时间。
5. **导入预览**：质量评分（0–100）与核对清单（✓ / ! / ×），填入前可知可能要改的地方。

---

## 我们不承诺什么

1. 智能导入**不能**保证任意排版、任意模板 100% 准确还原。
2. **不承诺**「上传即可投递、零人工修改」——导入后须按核对清单检查。
3. 复杂表格、多栏 Word、**扫描版 PDF**（无 OCR）常需导入后手动调整或换文字版文件。
4. 不保证 HR 一定打开链接或给出反馈（仅提供打开记录供候选人参考）。
5. 同一项目被拆条、奖项分行、校园经历误入项目区等，系统会持续优化，但**仍可能需人工修正**。

---

## 智能导入质量评分说明

| 等级 | 含义 | 建议 |
|------|------|------|
| **质量良好**（约 78 分以上且无严重项） | 字段较完整 | 填入后快速浏览履历与教育奖项即可 |
| **建议核对** | 可用作初稿 | 按核对清单逐项检查后再分享 |
| **需重点核对** | 存在明显缺口 | 填入后务必修正标红项，或部分区块手填 |

- 分数由**字段完整度 + 核对清单**综合估算，**不是** AI 模型自报的置信度。
- 即使分数较低，仍可点击「确认填入站点」，但请务必按清单修改后再发给 HR。

---

## 月租 / 续费说明（商业模式）

- 月租用于覆盖**服务器、域名、AI 解析**等持续成本。
- 到期后编辑权限或对外访问可能暂停；宽限期与数据保留期限以购买说明为准。
- 智能导入可按套餐包含**次数额度**，避免无限调用导致成本不可控。
